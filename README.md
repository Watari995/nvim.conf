# Neovim チートシート

> **`<leader>`** = **`Space`**
> **`<C-x>`** の `C` は **`Ctrl`** を意味する。例: `<C-v>` = `Ctrl + v`

## ファイルエクスプローラー (nvim-tree)

| キー | 動作 |
|------|------|
| `<leader>ee` | ファイルエクスプローラーをトグル |
| `<leader>ef` | 現在のファイルでエクスプローラーをトグル |
| `<leader>ec` | ファイルエクスプローラーを折りたたむ |
| `<leader>er` | ファイルエクスプローラーを更新 |
| `<leader>>` | エクスプローラーの幅を広げる（nvim-tree内） |
| `<leader><` | エクスプローラーの幅を狭める（nvim-tree内） |

---

## ウィンドウ / スプリット管理

| キー | 動作 |
|------|------|
| `<leader>sv` | 縦に分割 |
| `<leader>sh` | 横に分割 |
| `<leader>se` | 分割サイズを均等にする |
| `<leader>sx` | 現在のスプリットを閉じる |
| `<leader>sm` | スプリットを最大化 / 最小化 |

---

## タブ管理

| キー | 動作 |
|------|------|
| `<leader>to` | 新しいタブを開く |
| `<leader>tx` | 現在のタブを閉じる |
| `<leader>tn` | 次のタブへ |
| `<leader>tp` | 前のタブへ |
| `<leader>tf` | 現在のバッファを新しいタブで開く |

## Telescope (ファジーファインダー)

| キー | 動作 |
|------|------|
| `<leader>ff` | ファイルを検索 (cwd) |
| `<leader>fr` | 最近使ったファイルを検索 |
| `<leader>fs` | 文字列をライブ grep (cwd) |
| `<leader>fc` | カーソル下の文字列を検索 (cwd) |
| `<leader>ft` | TODO コメントを検索 |
| `<leader>fk` | キーマップを検索 |

### Telescope 内部操作

| キー | 動作 |
|------|------|
| `<CR>` | 現在のウィンドウで開く |
| `<C-v>` | 縦分割で開く (Ctrl + v) |
| `<C-x>` | 横分割で開く (Ctrl + x) |
| `<C-k>` | 前の結果へ移動 |
| `<C-j>` | 次の結果へ移動 |
| `<C-q>` | 選択をクイックフィックスリストへ |
| `<C-t>` | Trouble で開く ※新規タブではなく Trouble にカスタム済み |

## LSP (Go / TypeScript / Dart)

| キー | 動作 |
|------|------|
| `gR` | 参照一覧を表示 (Telescope) |
| `gD` | 宣言へジャンプ |
| `gd` | 定義へジャンプ |
| `gi` | 実装一覧を表示 (Telescope) |
| `gt` | 型定義一覧を表示 (Telescope) |
| `K` | ホバードキュメントを表示 |
| `<leader>ca` | コードアクション (ノーマル / ビジュアル) |
| `<leader>rn` | スマートリネーム |
| `<leader>D` | バッファの診断一覧 (Telescope) |
| `<leader>d` | 行の診断をフロート表示 |
| `[d` | 前の診断へ |
| `]d` | 次の診断へ |
| `<leader>rs` | LSP を再起動 |

## Git (gitsigns)

### Hunk 操作

| キー | 動作 |
|------|------|
| `]h` | 次の Hunk へ |
| `[h` | 前の Hunk へ |
| `<leader>hs` | Hunk をステージ |
| `<leader>hr` | Hunk をリセット |
| `<leader>hS` | バッファ全体をステージ |
| `<leader>hR` | バッファ全体をリセット |
| `<leader>hu` | Hunk のステージを取り消す |
| `<leader>hp` | Hunk をプレビュー |
| `<leader>hb` | 行の Blame を表示 (詳細) |
| `<leader>hB` | 行 Blame の表示をトグル |
| `<leader>hd` | Diff を表示 |
| `<leader>hD` | Diff を表示 (前のコミットと比較) |
| `ih` (ビジュアル/オペレータ) | Hunk をテキストオブジェクトとして選択 |

## 補完 (nvim-cmp)

| キー | 動作 |
|------|------|
| `<C-k>` | 前の候補を選択 |
| `<C-j>` | 次の候補を選択 |
| `<C-b>` | ドキュメントを上にスクロール |
| `<C-f>` | ドキュメントを下にスクロール |
| `<C-Space>` | 補完を起動 |
| `<C-e>` | 補完を中止 |
| `<CR>` | 選択を確定 |

## AI 補完 (Copilot)

| キー | 動作 |
|------|------|
| `<Tab>` | Copilot の提案を受け入れる |
| `<leader>at` | Copilot を有効 / 無効に切り替え |
| `<leader>as` | Copilot の状態を表示 |

初回は Neovim 起動後に `:Copilot setup` を実行して認証する。

---

## Surround (nvim-surround)

| キー | 動作 | 例 |
|------|------|----|
| `ys{motion}{char}` | 囲みを追加 | `ysiw"` → 単語を `"` で囲む |
| `ds{char}` | 囲みを削除 | `ds"` → `"` を削除 |
| `cs{old}{new}` | 囲みを変更 | `cs'"` → `'` を `"` に変更 |
| `S{char}` (ビジュアル) | 選択範囲を囲む | `S(` → 選択を `()` で囲む |

---

## Substitute (substitute.nvim)

| キー | 動作 |
|------|------|
| `<leader>r{motion}` | モーションで指定した範囲を置換 |
| `<leader>rr` | 行全体を置換 |
| `<leader>R` | カーソル位置から行末まで置換 |
| `<leader>r` (ビジュアル) | 選択範囲を置換 |

## Treesitter テキストオブジェクト

### テキスト選択 (`v` / `d` / `c` などのあとに使用)

| キー | 動作 |
|------|------|
| `a=` / `i=` | 代入式全体 / 代入の値部分 |
| `l=` / `r=` | 代入の左辺 / 右辺 |
| `a:` / `i:` | オブジェクトプロパティ全体 / 値 (JS/TS) |
| `l:` / `r:` | プロパティの左辺 / 右辺 |
| `aa` / `ia` | 引数全体 / 引数の中身 |
| `ai` / `ii` | 条件式全体 / 条件の中身 |
| `al` / `il` | ループ全体 / ループの中身 |
| `af` / `if` | 関数呼び出し全体 / 引数部分 |
| `am` / `im` | メソッド/関数定義全体 / 本体 |
| `ac` / `ic` | クラス全体 / クラスの中身 |

### 入れ替え (Swap)

| キー | 動作 |
|------|------|
| `<leader>na` | 引数を次と入れ替え |
| `<leader>pa` | 引数を前と入れ替え |
| `<leader>nm` | 関数を次と入れ替え |
| `<leader>pm` | 関数を前と入れ替え |
| `<leader>n:` | オブジェクトプロパティを次と入れ替え |
| `<leader>p:` | オブジェクトプロパティを前と入れ替え |

### コードブロック間の移動

| キー | 動作 |
|------|------|
| `]f` / `[f` | 次 / 前の関数呼び出し開始 |
| `]F` / `[F` | 次 / 前の関数呼び出し終了 |
| `]m` / `[m` | 次 / 前のメソッド定義開始 |
| `]M` / `[M` | 次 / 前のメソッド定義終了 |
| `]c` / `[c` | 次 / 前のクラス開始 |
| `]i` / `[i` | 次 / 前の条件式開始 |
| `]l` / `[l` | 次 / 前のループ開始 |
| `]s` | 次のスコープへ |
| `]z` | 次の折りたたみへ |

### モーションの繰り返し

| キー | 動作 |
|------|------|
| `;` | 直前のモーションを前方向に繰り返す |
| `,` | 直前のモーションを後方向に繰り返す |
